# Making AI Games

*Date: 19 May 2026*

I used clade (website) and `codex` on terminal to generate this game with prompt:

> Create an app in a fresh html file such that the user needs to tap one or more dots of different colors on the screen which disappear on tapping and new dots appear anywhere on the screen for a session of 1 min and then it gives the focus score and saves the focus score along with time and date so user can use this and analyse its score over time.
